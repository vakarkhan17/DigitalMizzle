# DigitalMizzle Cybersecurity Knowledge Base

A static, SEO-friendly prototype for DigitalMizzle: a premium cybersecurity education website with public knowledge-base pages, demo authentication, learner dashboard, downloads, and an admin console shell.

## Pages

- `index.html` - Home, hero, featured courses, blogs, tools, newsletter
- `courses.html` and `course-learning.html` - Course catalog and learning dashboard
- `blogs.html` and `blog-detail.html` - SEO blog listing and article layout
- `tools.html` and `tool-detail.html` - Cybersecurity tool directory and guide pages
- `downloads.html` - Resource downloads table
- `login.html`, `register.html`, `forgot-password.html`, `dashboard.html` - Demo user flow
- `admin.html` - Admin dashboard and creation workflow shell
- `sitemap.xml`, `robots.txt` - SEO support files

## Notes

The demo uses `localStorage` for account, verification, session, and progress behavior. A production version should replace this with a backend database, server-side authentication, email delivery, protected uploads, and role-based route middleware.
